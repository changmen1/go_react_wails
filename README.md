```bash

wails dev

```

```go
.
├── build/
│   ├── appicon.png
│   ├── darwin/
│   └── windows/
├── frontend/
├── go.mod
├── go.sum
├── main.go
└── wails.json
```

- /main.go - 主应用

- /frontend/ - 前端项目文件

- /build/ - 项目构建目录

- /build/appicon.png - 应用程序图标

- /build/darwin/ - Mac 特定的项目文件

- /build/windows/ - Windows 特定的项目文件

- /wails.json - 项目配置

- /go.mod - Go module 文件

- /go.sum - Go module 校验文件
